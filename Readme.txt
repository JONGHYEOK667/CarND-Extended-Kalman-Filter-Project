Clone/fork the project's template files from the project repository. (Note: Please do not submit your project as a pull request against our repo!)
Read the repo's README for more detailed instructions.
Clone the visualization and data generation utilities from the utilities repository.
Build an Extended Kalman Filter by applying the general processing flow as described in the previous lessons.
Ensure that your project compiles.
From the root of the repo:
mkdir build && cd build
cmake .. && make
./ExtendedKF
Test your code using the simulator!
Check your project against the project rubric. Make sure your RMSE values satisfy the thresholds listed in the rubric!
Submit your project for review. Be sure to include all files necessary to compile and run your project, including the CMakeLists.txt file.
